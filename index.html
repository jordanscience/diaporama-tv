⚙️ Configuration requise

Le site n'est pas encore relié à votre espace de stockage.

  1. Créez un projet gratuit sur supabase.com
  2. Créez un bucket media dans Storage
  3. Copiez l'URL du projet et la clé anon dans config.js
  4. Rechargez cette page

Toutes les étapes détaillées sont dans le fichier README.md.
